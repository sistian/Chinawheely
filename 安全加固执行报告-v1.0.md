### 4. 预订表单安全 (booking.html)

**新增5层安全防护**：

```javascript
// 1. 蜜罐检查
if (honeypot.value) { block submission }

// 2. 限流检查
if (!rateLimiter.check('booking')) { block submission }

// 3. 可疑模式检测
if (hasSuspiciousPattern(input)) { block submission }

// 4. 邮箱格式验证
if (!isValidEmail(email)) { block submission }

// 5. 输入长度检查
if (!checkLength(input)) { block submission }
```

**蜜罐字段**：
```html
<div style="position:absolute;left:-9999px;opacity:0;">
  <label for="website">Website</label>
  <input type="text" id="website" name="website" tabindex="-1" autocomplete="off">
</div>
```
